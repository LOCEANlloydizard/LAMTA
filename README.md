## LAMTA

**LAMTA** (**LA**grangian **M**anifolds **T**racking **A**lgorithm is a Python code designed to compute numerical particle trajectories within ocean current 2D fields and derive series of Lagrangian diagnostics to detect and track (sub)mesoscale ocean features.

### LAMTA manuscript and code

A manuscript is being developped to detail **LAMTA** utilities.

Tutorials are available in different Jupyter Notebooks to get started with **LAMTA**:
- Initializing and advecting particles in theoretical flows (Analytical_examples.ipynb)
- Initializing particles with ParticleSet (ParticleSet.ipynb)
- Setting up advection mode and parameter (Advection.ipynb)
- Dealing with periodic boundaries (Periodic_boundaries.ipynb)
- Ocean currents example (Ocean_examples.ipynb)
